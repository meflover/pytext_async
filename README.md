# Async Telegram Text Editor Bot

Асинхронный Telegram-бот для пошагового редактирования текста через диалоговый интерфейс  
с сохранением пользовательского состояния между сообщениями.

Проект демонстрирует работу с `aiogram 3`, асинхронной архитектурой, stateful-логикой  
и персистентным хранением данных.

---

## Основные компоненты

- `handlers/` — обработчики Telegram-событий
- `program.py` — логика текстового редактора и команд
- `session.py` — управление пользовательским состоянием (in-memory cache)
- `storage/` — персистентное хранилище (SQLite)
- `ratelimiter.py` — контроль частоты отправки сообщений

---

## Tech Stack

- Python 3.10+
- aiogram 3
- asyncio
- SQLite

---

## Features

- Асинхронная архитектура на базе `aiogram 3` (`asyncio`)
- Stateful text editing with per-user persistent state
- Поддержка текстовых команд:
  - `удали <слово>` — удалить слово из текста
  - `замени <a> на <b>` — заменить подстроку
  - `убери знаки` — удалить знаки пунктуации
  - `отзеркаль` — отразить текст
- Автоматическая нормализация текста:
  - удаление лишних пробелов
  - корректная пунктуация
- Хранение состояния пользователя в SQLite
- Rate limiting для защиты от флуда

---
